What is DOM and what is the purpose of DOM?

DOM stands for document object model and it is used to change HTML elements so that we can show our desired elements.

How will you select HTML elements using DOM? Name the DOM methods.

the following ways we can select HTML elemnts:

  • getElementsByTagName
  • getElementById
  • querySelector
  • querySelectorAll
  • getElementsByClassName

What is event bubble?

Event bubble occurs after an element take an event and that event bubble up or propagated to its parent ultill it reaches root element.

What is a callback function and why will you use it?

a callback function is a function that takes an argument from another function and later called from inside the outer function to finish an instruction. we use it when we have to run a function after a instruction is completed.
